# VoxelCpp
An experimental voxel game engine in Cpp (for learning)

# Dependencies and versions:
- KSC_Log 1.0 https://github.com/Hamderber/KSC_Log
- Vulkan 1.3 https://www.vulkan.org/
- GLFW 3.4 https://www.glfw.org/
- C++ 23
